정보 구조 설계 (IA)

웹사이트의 정보 구조 설계는 사용자가 정보를 쉽게 찾고 이해할 수 있도록 콘텐츠를 조직하고 분류하는 과정입니다. 본 프로젝트에서는 사용자 중심의 접근 방식을 통해 직관적인 사이트맵을 구축했습니다.

사이트맵 다이어그램 예시
홈
상품

신상품
이벤트
고객센터
마이페이지
장바구니

• '상품' 메뉴는 다양한 카테고리와 서브 카테고리로 세분화

• '마이페이지'는 주문 내역, 위시리스트 등 개인화된 정보를 한곳에 모음

사용자 흐름도 (User Flow)

사용자 흐름도는 사용자가 웹사이트 내에서 특정 목표를 달성하기 위해 거치는 일련의 단계를 시각적으로 표현한 것입니다. 본 프로젝트에서는 핵심 기능인 회원가입, 상품 탐색, 구매 결제 과정을 중심으로 사용자 흐름도를 설계했습니다.

1. 회원가입 사용자 흐름
시작: 웹사이트 접속
단계 1: '회원가입' 버튼 클릭
단계 2: 약관 동의
단계 3: 회원 정보 입력
단계 4: 이메일/휴대폰 인증
단계 5: '가입 완료' 버튼 클릭
종료: 회원가입 완료
2. 상품 탐색 및 구매 결제
단계 1: 상품 검색 또는 카테고리 선택
단계 2: 상품 목록에서 원하는 상품 클릭
단계 3: 상품 상세 페이지에서 옵션 선택
단계 4: 수량 조절 및 '장바구니' 버튼 클릭
단계 5: 장바구니 페이지에서 결제 정보 입력
단계 6: 결제 수단 선택 및 최종 확인
단계 7: 결제 완료 및 주문 확인
