DOM 조작 및 이벤트 처리

순수 JavaScript를 활용하여 웹 페이지의 동적인 요소를 구현하고 사용자 인터랙션을 처리했습니다.

DOMContentLoaded 이벤트: 문서 로드 후 스크립트 실행

이벤트 리스너: 탭 버튼 클릭 시 해당 탭 콘텐츠 활성화

이벤트 위임: 효율적인 이벤트 처리

탭 메뉴 구현 예시

document.addEventListener('DOMContentLoaded', () => {
    const tabs = document.querySelectorAll('.tab-button');
    const tabContents = document.querySelectorAll('.tab-content');

    tabs.forEach(tab => {
        tab.addEventListener('click', () => {
            // 모든 탭 버튼에서 'active' 클래스 제거
            tabs.forEach(item => item.classList.remove('active'));
            // 클릭된 탭 버튼에 'active' 클래스 추가
            tab.classList.add('active');

            const targetId = tab.dataset.target; // data-target 속성 값 가져오기

            // 모든 탭 콘텐츠 숨기기
            tabContents.forEach(content => content.style.display = 'none');
            // 클릭된 탭에 해당하는 콘텐츠 보이기
            document.getElementById(targetId).style.display = 'block';
        });
    });

    // 초기 로드 시 첫 번째 탭 활성화
    if (tabs.length > 0) {
        tabs[0].click();
    }
});

페이지 새로고침 없이도 콘텐츠를 전환하여 사용자 경험을 개선했습니다.

비동기 통신(AJAX)

fetch API를 사용하여 서버와 비동기적으로 데이터를 주고받는 기능을 구현했습니다.

비동기 데이터 요청: 서버에서 추가 데이터 가져오기

동적 콘텐츠 업데이트: 페이지 새로고침 없이 DOM 업데이트

오류 처리: 네트워크 문제 및 서버 응답 검사

"상품 더보기" 기능 구현 예시

document.getElementById('load-more-button').addEventListener('click', async () => {
    const currentPage = parseInt(document.getElementById('product-list').dataset.page);
    const nextPage = currentPage + 1;

    try {
        const response = await fetch(`/api/products?page=${nextPage}`);
        if (!response.ok) {
            throw new Error(`HTTP error! status: ${response.status}`);
        }
        const newProducts = await response.json();

        const productList = document.getElementById('product-list');
        newProducts.forEach(product => {
            const productItem = document.createElement('div');
            productItem.classList.add('product-item');
            productItem.innerHTML = `
                <img src="${product.image}" alt="${product.name}"/>
                <h3>${product.name}</h3>
                <p>${product.price}원</p>
            `;
            productList.appendChild(productItem);
        });

        // 페이지 번호 업데이트
        document.getElementById('product-list').dataset.page = nextPage;
    } catch (error) {
        console.log('Fetch error:', error);
    }
});

사용자는 페이지를 새로고침할 필요 없이 더 많은 상품을 탐색할 수 있어 로딩 시간을 단축하고 원활한 사용자 경험을 제공했습니다.
