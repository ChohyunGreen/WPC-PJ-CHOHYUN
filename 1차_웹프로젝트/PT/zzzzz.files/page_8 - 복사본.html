시맨틱 HTML 마크업

인터넷 쇼핑몰 웹사이트는 정보의 구조화와 검색 엔진 최적화(SEO), 그리고 웹 접근성 향상을 위해 시맨틱 HTML 마크업을 적극적으로 활용했습니다.

주요 시맨틱 태그 활용

<header>
<nav>
<main>
<article>
<footer>

주요 이점

검색 엔진 최적화(SEO)

검색 엔진이 웹 페이지의 내용을 더 정확하게 이해하고 색인화하는 데 도움을 줍니다.

웹 접근성 향상

스크린 리더와 같은 보조 기술을 사용하는 사용자들에게도 페이지 구조를 명확하게 전달하여 웹 접근성을 높였습니다.

콘텐츠 의미 명확화

각 콘텐츠의 의미를 명확히 전달하여 사용자 경험을 개선하고, 쇼핑몰의 정보 구조를 더 잘 이해할 수 있게 도와줍니다.

상품 목록 페이지의 시맨틱 구조







상품 목록 - [쇼핑몰 이름]


    <header>
        <h1>[쇼핑몰 이름]</h1>
        <nav aria-label="메인 메뉴">
            <ul>
                <li><a href="/category/new">신상품</a></li>
                <li><a href="/category/best">베스트</a></li>

            <input type="search" placeholder="상품 검색"/>
            <button type="submit">검색</button>
        </div>
    </header>
    
    <main>
        <section aria-labelledby="product-list-heading">
            <h2 id="product-list-heading">모든 상품</h2>
            <div class="product-grid">

                    </a>
                </article>

                        <img src="product-image-2.jpg" alt="상품명 2"/>

                        <p class="price">25,000원</p>
                    </a>
                </article>
                
            </div>
        </section>
    </main>
    
    <footer>
        <nav aria-label="푸터 메뉴">
            <ul>
                <li><a href="/about">회사 소개</a></li>

            </ul>
        </nav>
        <p>© 2025 [쇼핑몰 이름]</p>
    </footer>

시맨틱 마크업은 검색 엔진이 페이지 내용을 이해하고, 스크린 리더가 내용을 읽어주는데 도움이 됩니다.
