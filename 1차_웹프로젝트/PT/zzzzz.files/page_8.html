7. HTML 마크업

시맨틱 HTML 마크업

정보의 구조화와 검색 엔진 최적화(SEO)에 반응하도록 구성하였으며, 웹 접근성 향상을 위해 시맨틱 HTML 마크업을 적극적으로 활용했습니다.

주요 시맨틱 태그 활용

<header>
<nav>
<main>
<article>
<footer>

마크업 중점 사항

검색 엔진 최적화(SEO)

검색 엔진이 웹 페이지의 내용을 더 정확하게 이해하고 색인화하도록 반영했습니다.

웹 접근성 향상

스크린 리더와 같은 보조 기술을 사용하는 사용자들에게도 페이지 구조를 명확하게 전달하여 웹 접근성을 높였습니다.

콘텐츠 의미 명확화

각 콘텐츠의 의미를 명확히 전달하여 사용자 경험을 개선하고, 쇼핑몰의 정보구조이해와 브랜드 홍보를 극대화하는데 더 잘 이해할 수 있도록 설계했습니다.
