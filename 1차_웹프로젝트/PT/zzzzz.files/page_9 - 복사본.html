BEM 방법론 & SASS 전처리기

본 프로젝트에서는 CSS 코드의 유지보수성과 재사용성을 높이기 위해 BEM(Block, Element, Modifier) 방법론을 적용하고, SASS(SCSS) 전처리기를 활용하여 스타일을 구조화했습니다.

클래스 이름의 충돌 방지
코드 가독성 향상
변수, 믹스인 활용
중첩 규칙 지원

SASS(SCSS) 변수 및 믹스인 활용 예시

// _variables.scss
$primary-color: #007bff;
$secondary-color: #6c757d;
$font-stack: 'Noto Sans KR', sans-serif;

// _mixins.scss
@mixin flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
}

// button.scss
.button {
  background-color: $primary-color;
  color: #fff;
  padding: 10px 20px;
  border-radius: 5px;
  font-family: $font-stack;
  @include flex-center;

  &--secondary {
    background-color: $secondary-color;
  }
}

Flexbox & Grid 레이아웃

다양한 화면 크기에 유연하게 대응하는 레이아웃 설계를 위해 Flexbox와 CSS Grid를 적극적으로 활용했습니다. Flexbox는 주로 1차원적인 레이아웃에 사용되었으며, CSS Grid는 2차원적인 레이아웃 구성에 활용되었습니다.

Flexbox: 1차원 레이아웃
Grid: 2차원 레이아웃
