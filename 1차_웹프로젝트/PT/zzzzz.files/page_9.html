8. CSS 스타일

주요 속성

Grid Layout을 활용한 컨텐츠 나열

Transition & Animation

Transform 효과

swiper슬라이드와 제이쿼리를 활용한 동적요소 표현

CSS 코드의 유지보수성과 재사용성을 높이기 위해 BEM(Block, Element, Modifier) 방법론을 적용하고, SASS(SCSS) 전처리기를 활용하여 스타일을 구조화

/* CSS3 스타일링 */
:root {
--blackcolor: #000;
--pointcolor: #9333EA;
--width: 1200px;
}
.wrap {
max-width: var(--max-width);
margin: 0 auto;
padding: 0 20px;
}
.product-card {
transition: transform 0.3s;
}
