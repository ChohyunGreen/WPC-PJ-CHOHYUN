모바일 우선 접근법

이 프로젝트에서는 '모바일 우선(Mobile-First)' 접근법을 채택하여 디자인 및 개발을 진행했습니다.

  • 작은 화면의 모바일 기기에서부터 시작하여 점진적으로 더 큰 화면에 맞춰 디자인을 확장
  • 모바일 사용자가 전체 웹사이트 트래픽의 절반 이상을 차지하는 현대 웹 환경에서 사용자 경험을 최적화
  • 핵심 콘텐츠와 기능을 우선적으로 고려하여 불필요한 요소를 줄이고, 로딩 속도를 개선

미디어 쿼리 적용

다양한 디바이스 환경에서 최적의 사용자 경험을 제공하기 위해 CSS 미디어 쿼리(Media Query)를 적극적으로 활용했습니다.

기본 스타일 (모바일 우선)

width: 100%, padding: 15px, flex-direction: column

태블릿 (min-width: 768px)

flex-direction: row, justify-content: space-between, 2열 배치

데스크톱 (min-width: 1024px)

max-width: 1200px, margin: 0 auto, 4열 배치
